# Study Rules

## 1. 학습 방식

- 모든 학습은 "문제"에서 시작한다.
- 코드로 검증한다.
- 결과를 분석한다.
- 면접 답변으로 정리한다.

---

## 2. 금지 사항

- 개념 암기만 하는 학습
- 코드 없는 이론 정리
- 결과 없는 실습

---

## 3. Git 사용 원칙

- 모든 실험 결과는 커밋으로 기록
- 커밋 메시지는 "Problem → Result" 구조
- 세션 단위로 태그 관리

---
