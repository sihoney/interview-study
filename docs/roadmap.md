# Java Interview Startup Study — Roadmap

## Project Goal

이 프로젝트는 Java 면접 대비를 위한 단순 지식 정리가 아니라  
**시스템 사고 기반 문제 해결 훈련**을 목표로 한다.

---

## Learning Philosophy

암기 ❌  
문제 해결 ⭕  
시스템 사고 ⭕  
실무 대응 ⭕  
면접 합격 ⭕

---

## Study Flow

각 회차는 동일한 구조를 따른다.

Problem → Theory → Design → Implementation → Analysis → Interview

---

## Session Plan

| Session | Topic |
|--------|-------|
01 | Core Java Fundamentals |
02 | Backend Architecture |
03 | Spring Internals |
04 | System Design |
05 | Database & Tuning |
06 | Distributed System |
07 | Performance Engineering |
08 | Final Mock Interview |

---

## Final Outcome

- 실무에서 즉시 적용 가능한 사고력 확보
- 면접 질문에 시스템 관점으로 대응 가능
- 장기 성장 가능한 개발자 기반 구축
