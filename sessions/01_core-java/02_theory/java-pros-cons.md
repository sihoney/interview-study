# Java Pros & Cons

## Java 특징

Java는 객체지향 프로그래밍 언어이다.  
기본 자료형을 제외한 모든 요소들이 객체로 표현되며,  
캡슐화, 상속, 다형성이 잘 적용된 언어이다.

## 장점

- JVM 위에서 동작 → 운영체제 독립성
- Garbage Collector에 의한 자동 메모리 관리
- ClassLoader를 통한 동적 로딩 지원

## 단점

- JVM 기반 실행으로 네이티브 언어보다 상대적으로 느림
- 다중 상속 불가, 엄격한 타입 시스템
- 바이트코드 → 기계어 번역 단계로 인한 오버헤드
